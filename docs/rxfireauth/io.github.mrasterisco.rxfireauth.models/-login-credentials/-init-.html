LoginCredentials(idToken: String, accessToken: String?, fullName: String?, email: String, password: String?, provider: Provider, nonce: String?)

This class represents a set of credentials used to perform a sign in with a specific authentication provider.

Instances of this class are returned when a recoverable error, such as MigrationRequiredException, occurs during a sign in.

You shouldn't need to to inspect the content of this class. Its main purpose is to temporary store credentials in order to continue the login action when your client has handled the error.
