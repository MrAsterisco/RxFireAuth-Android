updatePassword

fun updatePassword(newPassword: CharSequence): Completable

Update or set the password of the currently signed in user.

If the user does not have a password among their UserData.authenticationProviders, this function will create a new provider using the user email and the specified password. This will basically link the Email & Password authentication to the user. If the user already has Provider.Password as an authentication provider, this function will simply update their password.
