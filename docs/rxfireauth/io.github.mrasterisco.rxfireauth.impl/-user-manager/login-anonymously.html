loginAnonymously

fun loginAnonymously(): Completable

Login an anonymous user.

You can use this method to create an anonymous user on the server.

The resulting Completable will emit AlreadyLoggedInException if there is already a non-anonymous user logged-in. It will also emit AlreadyAnonymousException if there is already an anonymous user logged-in.

Return
A Completable action to observe.
