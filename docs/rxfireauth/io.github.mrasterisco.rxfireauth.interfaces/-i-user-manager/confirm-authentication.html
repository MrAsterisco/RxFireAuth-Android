confirmAuthentication

abstract fun confirmAuthentication(email: CharSequence, password: CharSequence): Completable

Confirm the authentication of the passed credentials with the currently signed in user.

You need to confirm the authentication of a user before performing sensitive operations, such as deleting the account, associating a new login provider or changing the email or password.

To confirm the authentication with a login provider (such as Google), use the appropriate method in the "confirmAuthenticationWith" family, or confirm the authentication by other means and then call confirmAuthenticationWithCredentials.
