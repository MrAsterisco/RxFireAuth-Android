IUserManager

interface IUserManager

This interface defines the public APIs of the main wrapper around the Firebase Authentication SDK.

When using the library in your code always make sure to reference this interface instead of the default implementation UserManager, as this interface will always conform to Semantic Versioning.

All methods of this interface are wrapped inside a Rx object that will not execute any code until somebody subscribes to it.
