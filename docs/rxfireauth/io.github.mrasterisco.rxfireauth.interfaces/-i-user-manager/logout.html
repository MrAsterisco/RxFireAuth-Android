logout

abstract fun logout(resetToAnonymous: Boolean): Completable

Sign out the currently logged-in user.

Using resetToAnonymous, you can make sure that there is always a user signed in; in fact, if the parameter is set to true, this function will call loginAnonymously immediately after the logout operation has completed.

Parameters

resetToAnonymous - If true, after having logged-out successfully, this function will immediately sign in a new anonymous user.

Return
A Completable action to observe.
