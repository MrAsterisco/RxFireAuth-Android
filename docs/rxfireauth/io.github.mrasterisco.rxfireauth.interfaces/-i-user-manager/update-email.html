abstract fun updateEmail(newEmail: CharSequence): Completable

Update the email of the currently signed in user.

All users have an email address associated, even those that have signed in using a login provider (such as Google). Keep in mind that some login providers may return a relay email which may not be enabled to receive messages.

Parameters

newEmail - The new email address.
